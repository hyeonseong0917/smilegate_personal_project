# 스마일게이트 개인 과제
## 목표 설정
- 프로젝트 선택: PROJECT 2 인증 시스템
- 구현 목표
  - 가입, 로그인 페이지 -> API로 구현
  - 유저 관리 페이지 -> 웹 사이트로 확인
  - 게임 리뷰 및 평점 확인
  - RDBMS DB와 연동 -> 일단 sqlite에서 연동
  - Password Encryption -> Option
  - 캐시(Option)
  - 이메일 인증 및 비밀번호 찾기 (Option)
- 기술적으로 해보고 싶은 목표
  - Django에서 JWT를 이용한 인증 처리
    - 사용자 등록
    - 등록된 사용자로 토큰 요청 및 생성
    - admin을 제외하고는 review 수정 불가
